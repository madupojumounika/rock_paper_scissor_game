# Rock, Paper & Scissors 🎮

A simple and interactive **Rock, Paper & Scissors** game built with **HTML, CSS, and JavaScript**. Play against the computer and keep track of your score in real-time.

---

## 📝 Project Overview

This game allows users to:

- Play Rock, Paper, Scissors against the computer.
- View scores for both user and computer.
- Get instant feedback on the result of each round (win, lose, or draw).
- Enjoy a visually appealing UI with background images and hover effects.

---

## ⚡ Features

- **Interactive gameplay**: Click on rock, paper, or scissors to make your move.  
- **Score tracking**: Keep track of your wins and the computer’s wins.  
- **Dynamic messages**: Colored messages indicate game results:
  - Green → You win  
  - Red → You lose  
  - Blue → Draw  
- **Responsive design** and hover effects.

## 🕹 How to Play

1. Open `index.html` in a web browser.  
2. Click on one of the three options: **Rock**, **Paper**, or **Scissors**.  
3. The computer will randomly choose its move.  
4. The message box will show the result: win, lose, or draw.  
5. Scores are updated automatically.
